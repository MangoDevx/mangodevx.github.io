Projects Here are some of my recent projects that I have completed on the side.

GameTracker

GameTracker is an all-purpose application that can track your applications and how long you spend on each application. This data is taken and then presented on a website on your machine and displays it in graphs.
Check it out!

Mangopad

Mangopad is a metro style text editor with some of my favorite features. This includes auto-saving, printing, character count, word count, and a nice mango logo.
Check it out!

SmartPixel

SmartPixel is an application that can take any existing image and redraw it using the generated color palette. It doesn't matter if you pick 5 colors or 500, it will redraw the image. You can generate 2 types of palettes, a pastele palette or a grayscale palette.
Check it out!

Shout

Shout is a mobile application that was made for a Software Engineering project. Shout allows cloud messaging to other users within a 5 mile radius.
Check it out!
